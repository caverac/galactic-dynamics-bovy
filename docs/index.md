# Dynamics and Astrophysics of Galaxies - Solutions

Solutions to problems from **Dynamics and Astrophysics of Galaxies** by Jo Bovy.

## Book Reference

> Bovy, J. (2026). *Dynamics and Astrophysics of Galaxies*. Princeton University Press.
>
> - Online version: [galaxiesbook.org](https://galaxiesbook.org/)
> - Publisher: [Princeton University Press](https://press.princeton.edu/books/paperback/9780691212876/dynamics-and-astrophysics-of-galaxies)

## About This Project

This repository contains Python implementations and solutions to problems from the textbook. The solutions leverage [galpy](https://www.galpy.org/), the galactic dynamics library developed by the author of the textbook.
